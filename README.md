# Claude Panel for Obsidian

A native Obsidian desktop plugin that adds a VS Code Claude Code-like sidebar:
threads, attachments, proposal/diff review, and permissioned apply modes.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Plugin Artifacts

- `manifest.json`
- `main.js`
- `styles.css`
- `versions.json`
